1. Read the presentation file Sample Size and Portfolio Construction.pdf, where the results of analysis are presented.
2. Open the file Sample Size and Portfolio Construction.R.
3. Open menu Edit in your R shell.
4. Select Execute or Run All.
5. There will appear a chart window. Use chart menu in the R shell (Quartz) to move back and forth from Mean excess returns to Risks and Tracking errors.
6. The script file uses the libraries quantmod and XML. To install them type in your Console 
> install.packages(‘quantmod’)
> install.packages(‘XML’)
